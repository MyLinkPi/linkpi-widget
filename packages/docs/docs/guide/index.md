# 介绍

LinkPi Widget 是一个用于构建可视化配置的组件开发框架。它主要用于开发可在 LinkPi 平台中使用的自定义组件。

## 组件示例

![组件示例](/example.png)

![组件示例2](/example2.png)

如上图所示，每个组件包含三个主要部分：
- 预览区：展示组件在页面中的预览效果
- 设置面板：用于配置组件的属性和数据
- 渲染视图：最终在页面中渲染的实际内容

## 主要功能

- 可视化配置：通过设置面板轻松配置组件属性
- 实时预览：即时查看组件效果
- 数据联动：支持组件间的数据共享和通信

## 技术架构

框架由以下几个核心包组成：

- `@mylinkpi/widget-core`: 提供基础的组件定义和运行环境
- `@mylinkpi/widget-react`: React 组件库和开发工具
- `@mylinkpi/widget-cli`: 开发、构建和发布工具
